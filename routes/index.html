Welcome to the HelpDesk Issue Tracking API

This API can be utilized to create, update, delete and return data for the End User Issue Tracking Application.

Supported methods:

GET, POST, DELETE, PATCH

How to use:

You can use AJAX to call the API and will receive generated json data in return. If you are using jQuery, you can use the $.ajax() function shown below to get started.

$.ajax({
url: 'https://randomuser.me/api/',
dataType: 'json',
success: function(data) {
console.log(data);
}
});

Available routes

FETCH ALL KNOWN ISSUES

GET: https://testbongoapi.herokuapp.com/known-issues/

FETCH A SINGLE KNOWN ISSUE BY ID

GET: https://testbongoapi.herokuapp.com/known-issues/:issueId

CREATE A NEW KNOWN ISSUE

POST: https://testbongoapi.herokuapp.com/known-issues/

DELETE A SPECIFIC KNOWN ISSUE

DELETE: https://testbongoapi.herokuapp.com/known-issues/:issueId
